# Expense & Budget Visualizer

A modern responsive web application for tracking personal expenses, managing spending limits, and visualizing budget distribution with interactive charts.

## Features

### Core MVP Features
- Add transactions with item name, amount, and category
- Input validation for all required fields
- Scrollable transaction history list
- Delete transactions instantly
- Real-time total spending calculation
- Dynamic spending distribution chart using Chart.js
- Persistent local storage support

### Optional Enhancements Implemented
- Custom category creation
- Highlight transactions exceeding spending limit
- Dark / Light theme toggle
